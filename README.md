# Ye Olde Calculator

A console-based calculator application that performs basic arithmetic operations.
---

## Features
- Implements Shunting-Yard algorithm
- Supports +, -, *, /, ^, and parenthetical expressions

## Installation
```bash
git clone https://github.com/h-ordonez/ye-olde-calculator.git
cd path/to/ye-olde-calculator
```

## Usage
1. Open your preferred IDE
2. Run the program
3. At the console, enter a mathematical expression (e.g., _5+(2*3)_)
4. Press **Enter**
5. Try another expression or type "exit" to quit
